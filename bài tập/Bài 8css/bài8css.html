Border Color

Sytax: border-color:(top)(right)(bottom)(letf);
Example: border-color: #65C226 #7C200 #4493AD #DF127B;

Hear the top border is green(#85C226) color,
right border is yellow(#F7C200),
the bottom has blue(#4493AD)
and Left border is pink(#DF127B) in color.

Border Styles   Important  

This property must be used if you are using any of the Border properties.
Without using this border-style property you won't see any border at all.
Sytax: border-style:(top)(right)(bottom)(letf);
Example: border-style: solid dotted groove double;

Hear the top border (green) has a solid style,
right (yellow) one has a dotted,
bottom (blue) one has groove and
the left (pink) one has the double style.

Border Width

Sytax: border-width:(top)(right)(bottom)(letf);
Example: border-width: 5px 10px 15px 20px;

Hear the top (green) border's has a width is 5 pixel,
right (yellow) border is 10 pixel wide,
bottom (blue) is 15 pixel
and left (pink) is 20 pixel wide.

Border Radius

Sytax: border-radius:(top)(right)(bottom)(letf);
Example: border-radius: 0px 30px 70px 20px;

For displaying round comers Border Radius property is used.
IE9 and Opera use the same code. For Firefox a prefix"-moz" needs to be added before the code. And for Webkit Browers "-webkit- needs to be added before the code.
